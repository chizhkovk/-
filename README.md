# Проект для совместной разработки

Этот репозиторий создан для совместной разработки проектов между пользователем `chizhkovk` и ИИ-ассистентом Antigravity.

## Структура проекта
- Локальная разработка ведется на домашнем ПК.
- Версии отслеживаются через Git и хранятся на GitHub.
- Деплой и тестирование готовых контейнеров Docker осуществляются на удаленном сервере в Google Cloud.
